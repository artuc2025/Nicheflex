# Overview

## Purpose
This knowledge base stores insights, patterns, and decisions from AI-assisted development sessions.

## Structure
- **concepts/** - Atomic knowledge articles
- **connections/** - Cross-cutting insights linking 2+ concepts
- **qa/** - Filed query answers
- **problems/** - Problems encountered and solved
- **solutions/** - Reusable solution patterns
- **entities/** - People, tools, technologies
- **insights/** - Key learnings and observations

## How to Use
1. AI reads `index.md` to understand available knowledge
2. AI reads relevant articles when answering questions
3. New knowledge is added during sessions automatically or manually
