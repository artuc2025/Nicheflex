# NicheHeat — Locked Niche Profiles (FLEX-derived)

**Назначение:** единый источник правды для генерации. Каждый профиль задаёт структуру,
эмоциональный движок, арт-лок и анти-бан для своей ниши. Промпт скелета/сценария
подставляет нужный профиль по slug ниши.

**Два класса структур:**
- **DRAMA** (протагонист vs антагонист, withheld mystery, расплата): family-drama, true-crime, business-stories.
- **DOCUMENTARY** (withheld central question, раскрытие, рефлексия): history, space-science.

Источник DRAMA-спайна — твой `flex-longform-addon.md` BLOCK 1 (декодирован из 38× выброса).
DOCUMENTARY-спайн составлен по тому же принципу «один withheld-вопрос держит всё видео».

## 1. family-drama  *(DRAMA — эталон, из твоего addon)*

```
THEME:        Faceless English narrated family-betrayal & comeuppance
              (entitled relative wrongs narrator → narrator prevails)
LANGUAGE:     English   |   LENGTH: 9–15 min (~1500–2300 words)
PROTAGONIST:  First-person wronged narrator (faceless voice)
ANTAGONIST:   Entitled family member (MIL / parent / sibling / in-law) — per script
SETTING:      Contemporary domestic realism — homes, gatherings, weddings, funerals, workplaces
ART LOCK:     cinematic realistic, natural lighting, shallow depth of field,
              contemporary domestic setting, emotional documentary tone
              BANNED: cartoon, anime, Pixar, stylized 3D
EMOTIONAL ENGINE: injustice → simmering anger → anticipation → vindication → catharsis
VISUAL SIGNATURE: warm-to-cold contrast; tension cool/desaturated, payoff warm
ENDING/CTA:   comeuppance + 1-line moral + soft CTA + tease next story
```
**ARC (5-act):** HOOK+LOOP (open in the shock moment, end on a withheld mystery object) →
SETUP → ESCALATION ladder (4–6 escalating sabotages, false-relief re-hook) → TURN+REVEAL
(ally quietly prepared the weapon; recontextualises earlier slights; narrator stays clean) →
COUNTERATTACK WAVES (antagonist strikes back in escalating waves, each defeated & converted
to evidence) → COLLAPSE+CHERRY (tiered comeuppance: social→financial→total; end on the thing
she tried to steal, now the narrator's).

## 2. true-crime  *(DRAMA-adjacent — investigation/mystery)*

```
THEME:        Faceless English narrated true-crime / unsolved-case storytelling
LANGUAGE:     English   |   LENGTH: 10–18 min (~1700–2800 words)
PROTAGONIST:  Narrator-investigator (faceless); the victim as emotional anchor
ANTAGONIST:   The perpetrator / the unknown (the case itself is the adversary)
SETTING:      Real-world realism — towns, last-known locations, case timelines
ART LOCK:     cinematic realistic, desaturated cool grade, investigative documentary tone,
              archival-photo feel, dramatic low light
              BANNED: cartoon, anime, Pixar, stylized 3D, gore (advertiser-safe)
EMOTIONAL ENGINE: unease → curiosity → mounting dread → revelation → sober closure
VISUAL SIGNATURE: cold blues/greys; the "evidence" frame recurs; warm only in victim memory
ENDING/CTA:   the answer (or the haunting non-answer) + reflection + soft CTA + next case tease
```
**ARC (5-act):** HOOK+WITHHELD (open on the most chilling detail; pose the case question;
withhold) → VICTIM & NORMAL (humanise the victim, establish the ordinary before) →
INVESTIGATION ESCALATION (clues/leads ladder, each deepening the mystery; false-lead re-hook) →
TURN+REVEAL (the discovery/theory that recontextualises earlier details) → AFTERMATH
(consequences, the lingering question, justice-or-not). *Mystery object = the key piece of
evidence. "Counterattack waves" = escalating leads/dead-ends.*

## 3. business-stories  *(DRAMA-adjacent — hubris/scandal/collapse)*

```
THEME:        Faceless English narrated corporate scandal / rise-and-fall stories
LANGUAGE:     English   |   LENGTH: 10–16 min (~1700–2500 words)
PROTAGONIST:  Narrator-analyst; sometimes a whistleblower/founder as anchor
ANTAGONIST:   The hubris — greedy exec, fraudulent company, doomed strategy
SETTING:      Corporate realism — offices, boardrooms, factories, headlines, charts
ART LOCK:     cinematic realistic, clean corporate palette, documentary tone, archival/news feel
              BANNED: cartoon, anime, Pixar, stylized 3D
EMOTIONAL ENGINE: intrigue → ambition → tension → reckoning → lesson
VISUAL SIGNATURE: polished-then-cracking; sleek visuals decay as the collapse nears
ENDING/CTA:   the fall + the takeaway lesson + soft CTA + next case tease
```
**ARC (5-act):** HOOK+WITHHELD (open on the staggering number/headline of the collapse; pose
"how did it get here?"; withhold) → THE EMPIRE (the rise, the apparent success) → THE CRACKS
(escalating warning signs ignored; false-confidence re-hook) → TURN+REVEAL (the rot/fraud
exposed; recontextualises earlier "wins") → COLLAPSE+LESSON (tiered fall; the human cost;
the lesson). *Mystery object = the fatal flaw/document. "Waves" = escalating cover-ups.*

## 4. history  *(DOCUMENTARY)*

```
THEME:        Faceless English narrated history deep-dives ("what really happened")
LANGUAGE:     English   |   LENGTH: 12–20 min (~2000–3200 words)
NARRATOR:     Authoritative-but-curious documentary voice (faceless)
FOCUS:        An event/person/turning-point with a startling, under-told angle
SETTING:      Period realism — locations, artefacts, maps, recreations
ART LOCK:     cinematic realistic, period-accurate, museum/archival tone, painterly-realistic light
              BANNED: cartoon, anime, Pixar, stylized 3D, anachronisms
EMOTIONAL ENGINE: curiosity → growing stakes → tension → revelation → reflection
VISUAL SIGNATURE: warm aged tones; cold at the moment of consequence
ENDING/CTA:   the legacy/what-it-means + lingering thought + soft CTA + next topic tease
```
**ARC (5-act DOCUMENTARY):** HOOK+CENTRAL QUESTION (open on the startling fact/moment; pose
the withheld question "how/why did this happen?") → CONTEXT (the world before, the stakes) →
DEEPENING (escalating layers/forces that raise the question, each re-hooking) → REVELATION/TURN
(the cause/truth that recontextualises the earlier beats) → CONSEQUENCE & LEGACY (what it
changed, the reflection). *central_engine = the withheld historical question; escalation =
deepening forces, not counterattacks.*

```
NARRATOR:     Wonder-driven, precise documentary voice (faceless)
FOCUS:        A mind-bending phenomenon / open question / discovery
SETTING:      Cosmic realism — space, telescopes, data, scientific recreations
ART LOCK:     cinematic realistic, NASA/observatory tone, deep-space lighting, scientific-accurate

EMOTIONAL ENGINE: wonder → curiosity → tension → awe → contemplation
VISUAL SIGNATURE: deep blacks + luminous accents; scale-emphasising compositions

```
**ARC (5-act DOCUMENTARY):** HOOK+CENTRAL QUESTION (open on the staggering phenomenon; pose
"what is it / how can this be?"; withhold) → THE KNOWN (current understanding, the setup) →
THE DEEPENING (anomalies/escalating mystery that raise the question) → THE DISCOVERY/EXPLANATION
(the answer or leading theory that recontextualises) → THE IMPLICATION (what it means, the
wonder). *central_engine = the withheld scientific question; escalation = deepening anomalies.*

## SHARED ANTI-BAN LAYER (BLOCK 2 — mandatory for ALL profiles)

From your `flex-longform-addon.md` BLOCK 2. Every generation must obey:

1. **ORIGINALITY** — never narrate a real Reddit/forum/news post verbatim or near-verbatim.
   Invent names, city, specifics, structure. Inspiration ≠ transcription. (Content-ID + reused-content risk.)
2. **VARIATION across uploads** — rotate cold-open styles, antagonist/subject types, settings,
   narration rhythm. Channel-wide sameness is the #1 termination trigger.
3. **HUMAN EDITORIAL JUDGMENT** — deliberate framing, a genuine POV/insight beat.
4. **VISUAL EFFORT** — motion (b-roll, pan/zoom, jump cuts, overlays), never static slideshows.
5. **DISCLOSURE & SAFETY** — advertiser-friendly; no gratuitous violence, no real named private
   individuals, no defamation. Toggle synthetic-content disclosure where required.
6. **TITLE/THUMBNAIL HONESTY** — dramatic but true to the content.

The engine must REFUSE to generate anything violating #1 or #5 and steer the user back.
