# NicheHeat — MVP Document (PRD v1.2)

**Продукт:** платформа аналитики YouTube-ниш + генерация сценарных каркасов для faceless-криэйторов
**Рабочее название:** NicheFlex (заменить при выборе бренда)
**Автор:** Артур
**Дата:** июнь 2026
**Статус:** Draft → утверждается перед началом разработки

---

## 1. Vision (Зачем существует продукт)

Faceless-криэйторы тратят десятки часов на ручной ресёрч ниш и затем остаются один на один с вопросом «и что теперь снимать?». Существующие инструменты (1of10, NexLev, TubeLab, ApexVix) находят ниши — и бросают пользователя на самом дорогом шаге: исполнении.

**NicheFlex закрывает полный цикл: «нашёл нишу → понял, почему она работает → получил готовый сценарный каркас под неё».**

Одной фразой: *From niche to script in one click.*

Долгосрочная траектория (см. §13): единственная платформа полного жизненного цикла faceless-оператора — ниша → язык/рынок → хук → скрипт → запуск без бана.

## 2. Problem Statement

| Боль | Кто испытывает | Текущее решение | Почему оно плохое |
|---|---|---|---|
| Не знаю, какую нишу выбрать | Новички в faceless YouTube | Гайды, YouTube-гуру, ручной серфинг | Данные устаревшие, субъективные |
| Нашёл нишу, не понимаю механику успеха | Все сегменты | Ручной разбор конкурентов | 5–10 часов на нишу |
| Не могу превратить нишу в сценарий | Новички + студии | ChatGPT «в лоб», фрилансеры | Генерик-скрипты, риск бана YouTube за mass-produced AI контент |
| Поздно узнаю о растущих нишах | Опытные каналы | Подписки на конкурентов | Узнают, когда ниша уже перегрета |

## 3. Target Audience (ICP)

**Primary:** соло-криэйтор faceless-каналов, 20–40 лет, технически грамотный, уже потратил $0–500 на инструменты/курсы, доход с YouTube $0–3k/мес, хочет систематизировать выбор ниш. Платёжеспособность: $29–49/мес.

**Secondary:** микро-студии (2–5 человек), ведущие 5–20 каналов параллельно. Платёжеспособность: $89–149/мес (план Agency — после MVP).

**Anti-ICP (не обслуживаем в MVP):** крупные MCN, бренды, нефейсless-криэйторы с личным брендом.

## 4. Competitive Landscape

| Конкурент | Цена | Сильная сторона | Слабость (наше окно) |
|---|---|---|---|
| 1of10 | $49–89/мес | Outlier-поиск, бренд, $50k+ MRR | Нет шага «что делать дальше», нет фокуса на faceless |
| NexLev | пакеты + курс | База faceless-каналов | Перегружен, продаёт курс, нет генерации |
| TubeLab | подписка | Niche Analyzer | Только аналитика |
| VidIQ / TubeBuddy | $7.5–99/мес | Масштаб, дистрибуция | Generalist, копируют фичи медленно |

**Дифференциация NicheFlex:**
1. Единственный пайплайн «аналитика → сценарный каркас» (FLEX ENGINE: hook-loop, withheld mystery object, recontextualizing reveal и др. проверенные нарративные механики).
2. Anti-ban слой: каркасы спроектированы под политику YouTube 2026 против mass-produced AI контента (authenticity-маркеры, вариативность).
3. HEAT-score: собственная метрика «температуры» ниши (рост × монетизация ÷ конкуренция).

## 5. MVP Scope

### 5.1 In Scope (без этого не запускаемся)

**F1. Niche Radar (ядро)**
- Еженедельный автоскан YouTube Data API v3 (адаптация scan-youtube.mjs)
- Таблица ниш: HEAT-score, оценка RPM-диапазона, динамика просмотров за 7/30 дней, число активных каналов (прокси конкуренции), средний возраст каналов-лидеров
- Фильтры: язык, формат (long/shorts), уровень конкуренции, RPM-диапазон
- Карточка ниши: топ-5 outlier-видео, графики динамики

**F2. Niche Breakdown (дифференциатор, часть 1)**
- AI-разбор ниши: почему работает, какие нарративные механики используют лидеры, типовая структура видео, риски сатурации

**F3. Script Skeleton Generator (дифференциатор, часть 2)**
- Из карточки ниши → генерация сценарного каркаса через FLEX ENGINE (профиль ниши → hook → структура актов → reveal-точки → CTA)
- Лимиты по тарифу (см. монетизацию)

**F4. Auth + Billing**
- Email/Google OAuth (Supabase Auth)
- Lemon Squeezy (merchant of record — критично для Армении: налоги и приём платежей на их стороне)
- Тарифы: Free / Pro

**F5. Лендинг**
- Одна страница: проблема → демо (скрин Niche Radar) → прайсинг → FAQ
- Лид-магнит: «Топ-3 ниши недели» бесплатно по email (строим список рассылки с дня 1)

### 5.2 Out of Scope (осознанно НЕ делаем в MVP)

- Chrome-расширение
- Генерация видео/voiceover/тамбнейлов
- Трекинг собственного канала пользователя (OAuth к его каналу)
- Командные аккаунты, план Agency
- Мобильное приложение
- API для разработчиков
- Локализация интерфейса (только английский — Tier-1 аудитория)

> Правило: любая идея «а давай ещё…» попадает в backlog v2, не в MVP. MVP проверяет одну гипотезу: **готовы ли faceless-криэйторы платить за связку «ниша + скрипт-каркас»**.

## 6. Tech Stack

| Слой | Выбор | Обоснование |
|---|---|---|
| Frontend | **Nuxt 3 + TypeScript** | Основной стек Артура — нулевая кривая обучения |
| UI | Nuxt UI / Tailwind | Скорость сборки дашбордов |
| Backend | Nuxt server routes (Nitro) | Один репозиторий, один деплой |
| БД + Auth | **Supabase** (Postgres) | Auth, RLS, realtime из коробки, бесплатный тир на старте |
| Сборщик данных | Node.js worker (адаптация scan-youtube.mjs) + cron | Код уже существует |
| AI-слой | Anthropic API (Claude) | Niche Breakdown + Script Skeleton поверх FLEX-промптов |
| Платежи | **Lemon Squeezy** | Merchant of record, работает для Армении |
| Хостинг | Vercel / Railway (worker) | Деплой за минуты |
| Аналитика | Plausible / PostHog | Воронка: визит → trial → paid |
| Email | Resend | Лид-магнит + еженедельный дайджест |

## 9. KPI и критерии успеха

| Метрика | 30 дней | 60 дней | 90 дней |
|---|---|---|---|
| Email-лист (лид-магнит) | 300 | 800 | 1 500 |
| Регистрации Free | 150 | 400 | 800 |
| Платящие | 8 | 25 | 50 |
| MRR | ~$300 | ~$975 | ~$2 000 |
| Trial→Paid конверсия | — | ≥5% | ≥7% |
| Churn | — | — | <10%/мес |

**Kill-критерий:** если за 90 дней при выполненном плане дистрибуции (см. §11) платящих < 10 — гипотеза не подтверждена, пивот или закрытие. Фиксируем это сейчас, чтобы не тащить зомби-проект.

## 11. Go-to-Market (после запуска, 5 ч/день)

1. **Контент = демо (ежедневно, ~2 ч):** посты в X и Reddit (r/NewTubers, r/PartneredYoutube) с реальными данными из продукта: «ниша X: +340% за месяц, RPM ~$12, конкуренция низкая — скрин». Каждый пост — бесплатное демо.
2. **Собственный пайплайн (~1 ч):** faceless-видео «нашёл нишу с данными» через уже готовый видеопайплайн Артура — продукт рекламирует сам себя.
3. **Email-дайджест (еженедельно):** «Топ-3 ниши недели» — прогрев лид-листа до покупки.
4. **High-ticket сверху (по запросу):** done-for-you настройка канала $500–1 500 для 2–3 клиентов/мес — закрывает кассовый разрыв и даёт инсайты о болях.

## 12. Risks & Mitigations

| Риск | Вероятность | Митигация |
|---|---|---|
| Копирование фич крупными игроками (прецедент: VidIQ скопировал 1of10, −20% MRR) | Высокая | Скорость итераций + FLEX-слой как сложно копируемая глубина; строить email-лист как актив |
| Изменение квот/политики YouTube Data API | Средняя | Батчинг, кэширование, fallback на снапшоты; не нарушать ToS (только публичные данные) |
| Низкая конверсия Free→Pro | Средняя | Жёсткий пейволл на свежесть данных (главная ценность — актуальность) |
| Качество AI-каркасов «генерик» | Средняя | FLEX ENGINE с anti-ban слоем — уже спроектирован; человеческая валидация в бете |
| Один основатель = bus factor | Высокая | Максимальная автоматизация пайплайна данных; документация с дня 1 |
| Юр./налоговый риск приёма платежей из Армении | Низкая | Lemon Squeezy как merchant of record |

## 13. Product Trajectory — v2 Modules (после подтверждения гипотезы MVP)

Зафиксированная траектория развития. Каждый модуль — не новый продукт, а расширение жизненного цикла одного и того же клиента (faceless-оператора): *выбрал нишу → проверил язык/рынок → построил хук → запустил → не словил бан*. Модули включаются строго после достижения цели MVP (25 платящих / $975 MRR), по одному, в порядке приоритета.

| Приоритет | Модуль | Что делает | Почему именно мы | Эффект на бизнес |
|---|---|---|---|---|
| v2.1 | **Anti-Ban / Compliance Checker** | Пре-публикационный аудит канала/видео: риск демонетизации по политике YouTube против mass-produced AI контента, конкретные исправления (вариативность структуры, authenticity-маркеры, AI-disclosure) | FLEX BLOCK 2 (anti-ban слой) уже спроектирован; прямых конкурентов не обнаружено | Новый тариф +$20–30/мес или отдельный план; удержание клиента ПОСЛЕ запуска канала (снижение churn) |
| v2.2 | **Content Arbitrage Finder** | Поиск ниш, доказанных в EN, но пустых в других языках (ES, PT, HI, RU…): валидация спроса + оценка RPM целевого рынка | Мультиязычность основателя; расширение существующего сканера, а не новый пайплайн | Расширение TAM на не-англоязычных операторов; уникальная фича для контент-маркетинга |
| v2.3 | **Hook & Retention Intelligence** | Анализ транскриптов outlier-видео ниши: паттерны хуков первых 15 секунд, шаблоны под канал пользователя | Закрывает дыру, публично признанную конкурентами (никто не анализирует «почему скрипт работает») | Углубление ядра, рост ценности Pro-тарифа, повод поднять цену до $49 |
| v2.4 | **План Agency** | Мульти-канальный портфельный дашборд для студий (5–20 каналов): kill/scale решения, RPM по портфелю | Естественный апсейл по мере роста клиентов из соло в студии | Чек $99–199/мес, рост ARPU без роста числа клиентов |

**Парковка (отдельные ставки, не модули — рассматривать только при пивоте):**
- Channel Due Diligence / Valuation Tool («Carfax для покупки YouTube-каналов») — рынок сделок по faceless-каналам растёт (каналы ценятся в 2–3× из-за передаваемости), инструмента проверки перед покупкой нет; модель: разовый отчёт $49–99 + подписка для брокеров.

> Правило: ни один v2-модуль не начинается, пока MVP не достиг цели §8. Траектория зафиксирована, чтобы фичи v2 не просачивались в scope MVP.

## 14. Resolved Decisions (бывшие Open Questions)

**D1. Бренд: NicheHeat** (в коде допустим placeholder NicheFlex до покупки домена).
Обоснование: сильнейший брендовый паттерн категории — владеть словом-метрикой (прецедент: 1of10 владеет словом «outlier»). HEAT-score — собственная метрика продукта; бренд = метрика превращает каждое упоминание «HEAT 87» пользователями в бесплатный маркетинг. Суффикс «-flex» отбракован как затасканный и неинформативный.
☐ **Action (до недели 2):** проверить и купить nicheheat.com / nicheheat.io; беглая проверка товарного знака (USPTO/EUIPO поиск по «NicheHeat»).

**D2. Стартовые категории скана — 8 шт.** Логика портфеля: 5 high-RPM (для впечатляющих цифр на лендинге и в контент-маркетинге) + 3 high-volume (узнаваемость, «вижу своё»):
1. Финансы / инвестиции (RPM $15–45 — топ категории)
2. AI-новости и инструменты (растущая, бесконечный контент-пайплайн)
3. True crime / документалистика (прецеденты $80k+/мес)
4. Бизнес-истории / корпоративные расследования
5. История (документальный формат)
6. Семейные драмы / возмездие-наррация (инсайдерская экспертиза основателя)
7. Космос / наука
8. Хоррор-наррация / scary stories

**D3. Формат: только long-form в MVP.** Shorts — тумблер в неделю 5 (закрытая бета).
Обоснование: (а) ICP покупает ради AdSense RPM — long-form монетизируется на порядок лучше; (б) второй формат удваивает расход квоты YouTube API (лимит 10k units/день); (в) view velocity в Shorts искажает HEAT-score — метрика требует отдельной калибровки, это не задача недели 1.

**D4. Нейтив-вычитка лендинга: не требуется до запуска. Бюджет $0.**
Обоснование: аудитория — глобальные криэйторы, значительная доля ESL; AI-уровень английского достаточен для конверсии.
Триггер пересмотра: конверсия лендинга в регистрацию < 2% после запуска → копирайт-ревизия (~$100–150).
